# Check Design - A ferramenta com IA para análise de design UX/UI 

Este projeto se basea em uma landing page para usar uma ferramenta de inteligência artificial para analíse de design UX/UI, para suprir a demanda das empresas de contratação e procura imediata de profissionais nessa área.

Utilizei de tecnologias front-end e backend o NEXT.js, que supri a demanda de ambos os requisitos da aplicação.

Utilizarei o Vitest para testes automatizados.

Será utilizado também a API da Anthropic para o uso da Inteligência Artificial no projeto.

---

### Padrão de projeto da aplicação

O projeto segue as convenções propostas por programadores front-end, adotando a metodólogia proposta pela documentação da Vercel do NEXT.js 

---
 
Segue abaixo o setup para instalar localmente em sua máquina e rodar os comandos bases e de testes.

certifique-se de estar na pasta raiz do projeto no momento de rodar os comandos.

---

### Instalação - como baixar os arquivos
```bash
# instala os arquivos necessários da aplicação
npm install
```

---

### Iniciando o programa - como iniciar o projeto

```bash
# roda a aplicação em ambiente neutro o projeto 
npm run start
```

```bash
# roda a aplicação em ambiente de desenvolvedor
npm run dev
```

---

### Testes - rodando os testes da aplicação

```bash
# irá rodar os testes colocados dentro da aplicação
npm run test
```

```bash
# irá dar uma cobertura dos testes da aplicação 
npm run test:coverage
```

```bash
# irá dar uma interface gráfica a respeito dos testes da aplicação 
npm run test:ui
```

--- 

### Produção - fazendo a build do projeto

```bash
# esse comando irá rodar a build da aplicação para analisar o ambiente de produção
npm run build
```
